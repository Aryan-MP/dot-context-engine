Dot learns your codebase

Dot parses your codebase with AST-aware chunking - not naive text splitting. It understands functions, classes, dependencies, and how everything connects. Initial index takes minutes. Updates are instant.

~/acme-api
$ dot init
✓ 3,847 files indexed · 12,439 chunks · dependency graph built

Dot captures your decisions

Every architectural decision buried in commit messages, PR descriptions, and code comments gets surfaced, structured, and made queryable. Your codebase finally remembers why it is the way it is.

git log - decision mining

Dot injects context automatically

When you switch files, Dot assembles the most relevant context and injects it into every AI tool automatically. Semantic similarity, file proximity, recent changes, captured decisions - all combined, ranked, and delivered in under 100ms.

quick start

Ninety seconds
to a memory.

1

Install

One package. Python 3.11+. The embedding model downloads once, then everything is offline.

$ pip install dot-context[ml]
2

Initialize

Indexes your code, mines decisions from git history, wires up Claude Code and git hooks.

$ cd your-project && dot init
3

Forget about it

The daemon watches files, captures decisions, and serves context to every tool. Silently.

$ dot daemon start
why local-first

Your codebase is your company's most sensitive asset. Its memory shouldn't live on someone else's server. Dot runs on your machine, embeds on your CPU, and stores everything in a folder you can delete.
